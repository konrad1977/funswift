# Whats included

##### Monads
- `IO`
- `Deferred`
- `Reader`
- `Writer`
- `Changeable`
- `State`

#### Non monads
- `Predicate`
- `Endo`
- `Memoization`

#### Operators
- `<*>`	Applicatives 
- `>>-`	Bind 
- `>=>`	Fish
- `>>>`	Forward compose 
- `<<<`	Backward Compose 
- `|>`	Pipe

#### Protocols
- `Monoid`
- `Semigroup`

#### Extended swift types
- `Result`
	- zip
	- onSuccess	
	- onFailure
	- concat
